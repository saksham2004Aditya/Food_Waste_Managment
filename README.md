# Food Data Analytics Dashboard

## Project Overview

This project is a complete data analytics solution developed using Python, SQL, and Streamlit.

The workflow consists of:

1. Data Cleaning and Exploratory Data Analysis (EDA) using Python.
2. Data Querying and Business Insights using SQL.
3. Interactive Dashboard Development using Streamlit.

The goal is to transform raw food-related datasets into meaningful insights and present them through an easy-to-use dashboard.

## Tech Stack

* Python
* Pandas
* NumPy
* SQL
* Streamlit
* Plotly

## Project Workflow

### 1. Data Preprocessing & EDA

* Handle missing values
* Remove duplicates
* Analyze distributions
* Generate summary statistics
* Identify trends and patterns

### 2. SQL Analysis

* Data aggregation
* Filtering and sorting
* Joins and relationships
* KPI calculations
* Insight generation

### 3. Streamlit Dashboard

* Interactive visualizations
* Dataset filtering
* Business insights display
* User-friendly analytics interface

## Installation

Clone the repository:

git clone <repository-url>

Install dependencies:

pip install -r requirements.txt

Run the application:

streamlit run app.py
